             How to Decrypt
          

          
Use vault-pq on your local machine to decrypt .vault files downloaded from the builder.


          

            

              
1. Install vault-pq (requires Rust)

              
cargo install --git https://github.com/messecv3/vault-pq

            

            

              
2. Decrypt a file

              
vault decrypt -i agent.exe.vault -o agent.exe -k vault-operator-key

            

            

              
3. Batch decrypt all vault files in a directory

              
for f in *.vault; do vault decrypt -i "$f" -o "${f%.vault}" -k vault-operator-key; done
